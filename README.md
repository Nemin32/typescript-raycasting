# typescript-raycasting
Small raycasting application in your browser

## Controlls
| Button | Action |
|:-:|:-:|
| <kbd>Arrow keys</kbd> | Move |
| <kbd>m</kbd> | Show map |
| <kbd>s + Left/Right</kbd> | Strafing |
